# Student-Management-System
A Java-based desktop application for managing student records such as name, roll number, course, and
contact details.
This project is designed with a GUl using Java Swing and uses MySQL (or file system) for data storage.
Features-
Add new student
Update student records
Delete students
View student list
Search functionality

Tech Stack-

Technology      Description
Java           Programming Language
Swing          GUI Framework
MySql          Database
